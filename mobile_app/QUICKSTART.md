# UMKM Payment Validator - Flutter Mobile App

Mobile application for payment validation, inventory management, and book report OCR conversion.

## Quick Start

### 1. Start Backend
```bash
cd backend
python main.py
```

### 2. Run Flutter App
```bash
cd mobile_app
flutter pub get
flutter run
```

### 3. Login
- Username: `demo`
- Password: `demo123`

## Features

✅ Payment screenshot validation with OCR  
✅ Real-time dashboard with statistics  
✅ Inventory management with forecasting  
✅ Handwritten book report to Excel conversion  
✅ JWT authentication  
✅ Camera & gallery image upload  

## Tech Stack

- **Frontend**: Flutter 3.0+
- **State Management**: Provider
- **Backend**: FastAPI
- **Database**: SQLite
- **OCR**: EasyOCR
- **Authentication**: JWT

See full documentation in `README.md`
